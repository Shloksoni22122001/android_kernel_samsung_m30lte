# Samsung Galaxy M30 kernel (m30lte)

Android downstream Linux 4.4 kernel for the Samsung Galaxy M30, forked from
[akhil1999/android_kernel_samsung_m30lte](https://github.com/akhil1999/android_kernel_samsung_m30lte).

## postmarketOS port

The kernel now builds as a postmarketOS `linux-samsung-m30lte` package on the
`edge` channel. The package build is verified; booting and hardware enablement
still require testing on the phone.

See [the postmarketOS m30lte porting notes](docs/postmarketos-m30lte-port.md).
